# dive-into-tepui

地球低軌道(LEO)を舞台にした、軌道力学に基づくシューティングゲーム。

TypeScript + Three.js(WebGPUレンダラー)で構築。縮尺・時間スケールが正確な地球低軌道上で、ニュートン力学に基づいた軌道マニューバーを行いながら敵機を撃破する。

## 起動方法

### 必要環境

- Node.js
- WebGPU 対応ブラウザ(Chrome / Edge の最新版など)

### セットアップ

```bash
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

起動後、ブラウザで **http://localhost:8080** を開く。

### 本番ビルド

```bash
npm run build
```

`dist/` に成果物が出力される。

### 型チェック

```bash
npm run typecheck
```

## ゲームの遊び方

### 目標

画面右上 CONTACTS に表示される敵機5機をすべて撃破すればミッション完了。自機が高度80km以下まで落下すると大気圏で燃え尽きてゲームオーバー。

### 基本の流れ

1. `Tab` でターゲットを選択する
2. `F` で推進基準をターゲット基準に切り替え、接近する
3. 距離が離れている場合は `,` `.` でタイムワープして会合時間を短縮する
4. 射撃可能な距離(ワープ×4以下)まで来たら、リードマーカー(赤い ✛ 印)に照準を合わせて `Space` または左クリックで発砲する

### 操作方法

| キー | 動作 |
| --- | --- |
| `W` / `S` | 加速(プログレード)/ 減速(レトログレード) ※ターゲット基準では接近/離脱 |
| `A` / `D` | ノーマル / アンチノーマル ※ターゲット基準では左右 |
| `Q` / `E` | ラジアルイン / ラジアルアウト ※ターゲット基準では下/上 |
| `F` | 推進基準の切り替え(軌道基準 ⇄ ターゲット基準) |
| `I` / `K` | ピッチ(機首下げ / 機首上げ) |
| `J` / `L` | ヨー(左 / 右) |
| `U` / `O` | ロール(左 / 右) |
| `T` | RCS 回転制動の ON/OFF |
| `Tab` | ターゲット切り替え(近い順) |
| `Space` / 左クリック | 機関砲発射(タイムワープ×4以下でのみ可能) |
| `,` / `.` | タイムワープ 減少 / 増加 |
| `P` | 一時停止 |
| `H` | 操作方法ヘルプの表示切り替え |
| `R` | 再出撃(ゲーム終了後) |
| 右ドラッグ | カメラ視点の回転 |
| マウスホイール | カメラのズーム |

### HUD の見方

- **左上 SHIP STATUS**: ミッション経過時間、タイムワープ倍率、推進基準、RCS制動の状態
- **左中央 ORBIT**: 自機の高度・速度・遠地点/近地点高度・軌道傾斜角・周期
- **右上 TARGET**: 選択中ターゲットとの距離・接近速度・相対速度・HP
- **右中央 CONTACTS**: 残存する敵機の一覧(選択中のターゲットは強調表示)
- **画面中央のマーカー**: 機首方向(ボアサイト)、プログレード/レトログレード方向、ターゲット方向、リード(見越し)照準

## 現在の実装状況

- 地球低軌道での正確な二体軌道力学(RK4積分)
- 3次元の軌道マニューバー(軌道基準・ターゲット基準の両対応)
- 手動姿勢制御とRCSによる回転制動
- タイムワープ
- 機関砲によるシューティングと敵機の撃破
- 発砲ごとの薬莢排出、剛体物理演算による薬莢・破壊された宇宙船の挙動(ジャニベコフ効果を含む)
- 敵機全機撃破でのミッション終了、大気圏突入での機体喪失

大気抵抗、J2/潮汐摂動、太陽・月による三体摂動、敵AI、オーロラ演出などは未実装。詳細な設計方針は `dev.md`(人間のみが編集する文書)を参照。開発者向けの技術的な詳細は `CLAUDE.md` を参照。
